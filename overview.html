Genesis Awesome Theme
The Started Theme for Genesis Framework ideal for designers.


Este theme esta planeado para aquellos que solo deseen pasar más tiempo con las stylesheet que buscando funciones y agregando scripts

Esta basado en ee sample Theme v2.0.2 para la última version de Genesis Framework.

What´s include?

HTML5 Support

SASSy

Bootstrap Integration & Shortcode

You or your clients wants Bootstrap buttons or Jumbotron?. No tienes porque preocuparte más. GAT agrega bootstrap 3.2.0 y algunos shortcodes para que puedas utilizarlos dentro del editor de Wordpress y php.

Tons of Snippets

Los Snippets son grandiosos, nos ahorran bastante tiempo y nos ayudan a hacer que el sitio se vea como notros deseamos sin necesidad de mucho codigo. Por eso aquí estan los más populares.

Find it on: /lib/functions/snippets.php

Example:

//*** Customize the author box title
add_filter( 'genesis_author_box_title', 'custom_author_box_title' );
function custom_author_box_title() {
	return '<h4>Awesome history about me <h4>';
}
